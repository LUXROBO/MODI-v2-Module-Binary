## Feature

### Bootloader (e103)
1. 디스플레이 모듈이 업로드 또는 업데이트 중일 경우, 특정 이미지를 출력

### Display
1. Plug and Play 속도 증가
2. 사용자 코드 동작 시, 기본 이미지 출력

## Hotfix

### OS
1. 모듈을 장시간 켜 놓을 시, 모듈이 멈추는 오류 수정

### Battery
1. Plug and Play 동작 중, 베터리 모듈을 종료시켜도 디스플레이 모듈 화면에 베터리 모듈이 남아있는 오류 수정

### Motor
1. 원점이 잘못 적용되는 오류 수정
